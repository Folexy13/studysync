If Chrome AI is not available:

  1. Ensure you're using Chrome Canary or Dev (version 131+)
  2. Enable all required flags in chrome://flags
  3. Check chrome://components for "Optimization Guide On Device Model"
  4. Restart Chrome completely after enabling flags
  5. If still not working, use the Gemini API fallback
